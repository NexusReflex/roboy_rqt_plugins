### roboy_rqt_plugins
rqt gui plugins 

### installation
add the repo as a submodule to your catkin workspace:
```
git submodule add https://github.com/Roboy/roboy_rqt_plugins
```
build your workspace:
```
catkin_make
```
source your devel/setup.bash
### run it
fire up rqt, then add the plugins you need
